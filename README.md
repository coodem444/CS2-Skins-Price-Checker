\# CS2 Notion Price Updater



Automatiza la actualización de precios de skins de CS2 almacenadas en una base de datos de Notion.



\## Características



\- Steam Market

\- Skinport

\- CSFloat

\- Historial SQLite

\- Alertas Discord

\- Actualización automática de Notion



\## Instalación



```bash

pip install -r requirements.txt



Copia .env.example a .env y añade tus credenciales.



Ejecutar

python actualizar.py
